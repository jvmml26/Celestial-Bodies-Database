Celestial Bodies Database
Este projeto foi desenvolvido como parte da certificação de Relational Databases do freeCodeCamp.
Ele consiste em um banco de dados PostgreSQL que mapeia galáxias, estrelas, planetas e luas, utilizando chaves estrangeiras, restrições UNIQUE e tipos de dados variados.
Como rodar:
Instale o PostgreSQL.
Execute: psql -U postgres -f universe.sql
